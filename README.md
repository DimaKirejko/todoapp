# To Do App

Simple To Do App adapted for deployment in a product and local environment.
See [SwwaggerDoc](https://github.com/DimaKirejko/todoapp/blob/master/docs/swagger.yaml) and [.env.example](https://github.com/DimaKirejko/todoapp/blob/master/.env.example)

## Stack

**Language**: GO

**DB**: Postgresql (pgx & **Docker image**: migrate/migrate:v4.19.1)

**Transport**: HTTP

**Containerization**: Docker, Docker Compose
